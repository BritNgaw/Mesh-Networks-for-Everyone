# Mesh Networks for Everyone

## Test locally

```bash
python3 -m http.server 8000
```

Open http://localhost:8000 in your browser.

If you are on Windows and `python3` does not work, try:

```bash
python -m http.server 8000
```

## Files

- `index.html` - page shell
- `styles.css` - visual design
- `app.js` - interactions, animations, quizzes, maps
- `data.js` - outage, map, mesh examples, and survey data
- `assets/` - image files used by the case studies
